# WOW backend project using springboot
## Tech Stack
### Framework
* springboot2.X (backend framework)
* maven (package dependency manager)
* mybatis (ORM framework)
  * mybatis-plus (plugin)

### Storage
* mysql8.0
* redis

### Safety
* JWT token
* MD5 Encryption(password)

### Tools
* postman (API testing)

### Mock data
* javaFaker lib
* UUID

### API doc
* swagger2

## Backend layer ( MVC)
1. Controller
3. Business 
    * deliver **VO**
    * receive **DTO**
5. Service
6. Mapper 
    * **entity/DO**
