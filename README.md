# WOW backend project using springboot

### Framework
* springboot2.X (backend framework)
* maven (package dependency manager)
* mybatis (ORM framework)

### Storage
* mysql8.0
* redis

### Safety
* JWT token
* MD5 Encryption(password)

### Tools
* postman (API testing)

* UUID

### API doc
